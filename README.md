# my-java-project
Java dili ile kullanıcıdan bilgi alarak uçak rezervasyonu yapan proje

Bu projede java dili kullanılmıştır .  İçerisinde dört temel class  vardır  ve her class birçok değişken içermekte. 
CLASSLAR  : Ucak , Lokasyon , Musteri , Rezervasyon 
Bu proje uçak rezervasyonuyla ilgili olup gerekli alanlar yazılmıştır. Projede son olarak kullanıcıya ad soyad tarih ve uçuş ile ilgili gerekli tüm bilgiler alınmış olup kullanıcıya başarıyla uçak rezervasyonu yaptırılmıştır.
